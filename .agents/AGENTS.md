# project rules

- don't import anything external. all code must be standard java / libgdx because this is a student project.
- don't ever include yourself or mention the ai/model (e.g. gemini, antigravity) in the project, commits, or repo.
- write as much as you can in spanish.
- write simple code comments as long as they don't break conventions or make code errors.
- write comments in this exact format: `//comment` (no fancy designs, decorators, or dividers; keep them concise and lowercase).
- before committing or pushing anything, always run the proposed commit message by the user first for approval.
- keep responses and communication concise and lowercase.
